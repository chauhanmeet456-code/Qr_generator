# 🚀 QR Code Generator

A modern and responsive QR Code Generator built using **Python (Flask)**, **HTML**, **CSS**, and **JavaScript**.

Generate QR Codes instantly for URLs, Text, Email, Phone Numbers, and Wi-Fi with a clean Glassmorphism UI.

---

## ✨ Features

- 🌐 URL QR Code
- 📝 Text QR Code
- 📧 Email QR Code
- 📞 Phone Number QR Code
- 📶 Wi-Fi QR Code
- 🎨 Custom Foreground & Background Colors
- 📏 Adjustable QR Size
- 🌙 Dark / Light Mode
- 📋 Copy Input Button
- ⏳ Loading Animation
- ✅ Success Toast Notification
- 📥 Download QR as PNG
- 📄 Download QR as SVG
- 📱 Fully Responsive Design
- 🔒 Input Validation
- ⚡ Fast QR Generation

## ⚙️ Installation

### Clone Repository

```bash
https://github.com/chauhanmeet456-code/Qr_generator.git
```

### Go to Project Folder

```bash
cd QR-Code-Generator
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python app.py
```

Open your browser:

```
https://qr-generator-arec.onrender.com/
```
